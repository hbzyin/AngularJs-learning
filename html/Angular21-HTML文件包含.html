#### AngularJS--HTML文件包含

----->1.html文件暂时还不支持包含HTML文件功能；
----->2.AngularJS中，可以在HTML文件中包含HTML文件。
-----》3.服务器端包含：大多服务端脚本都支持包含文件功能 (SSI： Server Side Includes)。
-----》4.客户端包含 ：通常使用 http 请求 (AJAX) 从服务端获取数据，返回的数据通过使用 innerHTML 写入到 HTML 元素中。
1.angularJS包含指令：ng-include
1.1 包含html文件
1.2 包含angularjs文件
1.3 包含js文件
1.4 跨域包含
————————>默认情况下，ng-include指令不予许包含其他域名的文件，
